So here’s the Python Decorator guide.

In Python, functions are first-class objects. This means that functions can be passed around and used as arguments, just like any other object (string, int, float, list, and so on).

Inner Functions means defining function inside other functions. The inner functions are not defined until parent function is called. They are locally scoped to parent(是双亲函数的局部变量).

Simple Decorators Example

The so-called decoration happens at the last line. Name say_whee now points to the wrapper() inner function. 修饰器包裹函数，修改其行为。

Syntactic Sugar

You can using decorator in a simpler way with the @ symbol.

@my_decorator is just an easier way of saying say_whee = my_decorator(say_whee).
